# Compass

A portable React + TypeScript design system and AI agent workflow. Tokens, reference components, Storybook, Figma sync, and an integrated AI agent layer that ties Claude Code, Codex CLI, and Claude Design into the build/maintain loop. Designed to be forked and tuned for any new employer or client in under an hour. See `PLAN.md` for the master plan, decisions log, cross-checking protocol, and out-of-scope list.
